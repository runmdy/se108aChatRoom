# Start Server
```
$ cd team
$ npm start
```

## Use Websocket chatroom
* Open Chrome go to http://localhost:3000/chatroom.html
* Put your name and your message and click 

## Use firebase chatroom
* Open Chrome go to http://localhost:3000/chat.html
* Put your name and your message and click
